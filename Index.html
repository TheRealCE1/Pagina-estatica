Neural Network para Optimización de Citas (Diabetes)

Sistema en Python que interactúa con pacientes para evaluar síntomas y priorizar citas médicas. Enfocado en pacientes con diabetes; el modelo fue entrenado para recomendar la urgencia de la atención según las respuestas.

  • Modelo: Red neuronal implementada con TensorFlow/Keras
  • Preprocesamiento: NumPy y Pandas
  • Integración backend: lógica de priorización y enrutamiento de citas
  • Trabajo en equipo con control de versiones en GitHub
